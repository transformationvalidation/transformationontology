RML Mapping Scriptc back to ToC or Class ToC

IRI: https://data.europa.eu/m8g/transform-validate-ontology#RMLMappingScript

A specific type of transformation resource represented by an RML mapping file. RML - RDF Mapping Language is a generic mapping language, based on and extending the standard R2RML Through RML it is possible to express customized mapping rules in files (the resources that defines the transformation algorithm) from heterogeneous data structures and serializations to the RDF data model according to some ontologies.

stable
